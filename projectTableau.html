Tableau Project
Analysing COVID-19 data

The data used was from https://ourworldindata.org/covid-deaths. The data consists of various statistics about COVID-19 cases, deaths and vaccinations. In this project, the data was analysed to visualize how cases progressed. It is also interesting to look into how it evolved in different countries.

The worksheets below help achieve this goal. The data included are: global numbers of the cases and deaths, the percentage of cases in relation to deaths, the number of deaths per continent, the infection rate per country and the average number of cases per country.
