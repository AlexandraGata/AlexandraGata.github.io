MongoDB Project
Manage a music store inventory

Scenario

The scenario created for the database is a store that sells vinyl, CDs and other related products.

The store is constantly receiving supplies and selling products. Database creation aims to be able to keep track of stock. First, the store buys stock from one or more suppliers, and the quantity may vary between them. Contact details must be saved for each company.

Customers buy products in various quantities. Customer data is saved in case of future contact. In addition, the delivery address is required.

Within the database, a "Products" collection will be created. Documents in this collection will be composed of specific attributes (name, music genre, artist, product type, price). Additionally, each will have two arrays of objects: sales and supplies.

The project was created using variable names and data in Romanian. This is because it was part of the "Advanced Databases" course of my bachelor's.

Queries

How many products were sold on 07/01/2023.

db.produse.countDocuments({"vanzari.dataVanzare":ISODate("2023-01-09T00:00:00.000Z")})

What is the current stock of the product with id 7?

 
                                            db.produse.aggregate([{ $match: { _id: 7 } }, 
                                            { $addFields: { totalCantitateAprovizionare: { $sum: "$aprovizionari.cantitate" }, totalCantitateVanduta: { $sum: "$vanzari.cantitate" } } }, 
                                            { $project: { denumire: "$denumire", Stoc: { $subtract: ["$totalCantitateAprovizionare", "$totalCantitateVanduta"] } } }])

All the products that were bought by clients that are not from Cluj-Napoca.

 db.produse.find({"vanzari.client.localitate":{$ne:"Cluj-Napoca"}},{denumire:1}).sort({denumire:1})

All the companies that restocked.

 db.produse.distinct("aprovizionari.firma")

How many items were stocked and how many were sold for product with id 9.


                                            db.produse.aggregate([{ $match: { _id: 9} }, 
                                            { $addFields: {totalCantitateAprovizionare:{ $sum: "$aprovizionari.cantitate" },totalCantitateVanduta:{ $sum: "$vanzari.cantitate" }}}, 
                                            { $project : { denumire : "$denumire" , Aprovizionare:"$totalCantitateAprovizionare" ,Vandut:"$totalCantitateVanduta"}}])

The product with the highest price,

db.produse.find({tipProd:"vinil"},{denumire:1,pret:1}).sort({pret:-1}).limit(1)

What was the profit from all the products with id 9 sold.

 
                                            db.produse.aggregate([{ $match: { _id: 9 } }, 
                                            { $addFields: { totalCantitateAprovizionare: { $sum: "$aprovizionari.cantitate" }, 
                                            totalCantitateVanduta: { $sum: "$vanzari.cantitate" } } }, 
                                            { $project: { denumire: "$denumire", venit:{$multiply:["$pret", 
                                            { $subtract: ["$totalCantitateAprovizionare", "$totalCantitateVanduta"]}]} } }])
